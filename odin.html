The Odin Project

Zielstrebend

In der WISS-Schule wird nicht viel über die Programmierumgebung selbst (wie zB. Ubuntu) unterrichtet. Um sich in einer solchen zurecht zufinden, passt "The Odin Project" hervorragend als Lernunterstützung. "The Odin Project" ist eine Online Learning-Plattform, auf welcher man bei null anfängt. Der Lernprozess beginnt mit dem Umgang im Betriebssystem Xubuntu und folgt mit diversen Projekten, bei welchen HTML, CSS und Javascript angewendet wird.

Einige Projekte finden Sie in meinen GitHub repositories.
